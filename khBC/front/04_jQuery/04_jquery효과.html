효과

페이지에 애니메이션 효과를 만들기 위한 함수

사용방법 : 객체.함수이름([speed],[easing],[callback]);

speed : 실행속도(ms), easing : 변경되는 지점별 속도, callback : 해당함수 실행 후 동작할 함수를 설정

easing 옵션은 기본 jquery 로는 사용이 불가능하고, jquery-ui Library를 추가로 설치해야 사용 가능.

animate() : 현재 css 속성을 설정 속성으로 차츰 변경하는 효과를 주는 함수

객체.animate({객 체 형 식 으 로 CSS 속 성 },[speed],[easing],[callback]);

easing 옵션은 jquery-ui를 추가로 설치해야 사용 가능

모든 css 속성이 시간에 의한 변화가 일어나진 않음(jquery-ui를 추가로 설치하면 더 많은 속성변화가 가능)
